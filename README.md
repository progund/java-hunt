# java-hunt
An implementation of Silly-Hunt in Java

[JavaDoc](https://progund.github.io/java-hunt/index.html?se/itu/hunt/package-summary.html)

Build:
```Bash
$ ./build.sh
```

Run text ui version:
```Bash
$ ./run.sh --text
```

Run GUI version 1:
```Bash
$ ./run.sh --gui
```

Run GUI version 2:
```Bash
$ ./run.sh --gui2
```

Generate JavaDoc:
```
$ ./make_doc.sh
```
